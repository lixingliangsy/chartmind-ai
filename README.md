# ChartMind AI

Generate beautiful charts from natural language descriptions. Support for bar, line, pie, and more chart types.

## Features

- 📊 **Multiple Chart Types**: Bar, Line, Pie, Scatter, and more
- 🤖 **AI-Powered**: Natural language to charts
- ⚡ **Instant Generation**: Charts in seconds
- 💾 **Export Options**: PNG, SVG, PDF

## Quick Start

### 1. Install Dependencies

```bash
npm install
```

### 2. Run Development Server

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to see the app.

### 3. Usage

1. Enter your data in JSON format
2. Select chart type
3. Click "Generate Chart"
4. Download your chart
