# ChartMind AI - AI Chart Generator

> **One-line pitch**: Turn data into stunning charts in seconds with AI

---

## Quick Start

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

---

## Features

- 🤖 **AI-Powered**: Describe your chart in natural language
- 📊 **20+ Chart Types**: Bar, Line, Pie, Scatter, Heatmap, and more
- 🎨 **Beautiful Templates**: Academic, Business, Social Media presets
- 📄 **Code Export**: Python (matplotlib/seaborn), R, JavaScript
- 🏆 **High-Resolution**: Up to 1200 DPI for print
- 🔒 **Privacy-First**: Your data stays in the browser (optional)

---

## Tech Stack

- **Frontend**: Next.js 15 + React 19 + TypeScript
- **Styling**: Tailwind CSS + shadcn/ui
- **Charting**: D3.js + Recharts
- **AI**: OpenAI API + DeepSeek API
- **Export**: Puppeteer (PNG/SVG/PDF)
- **Deploy**: Vercel
